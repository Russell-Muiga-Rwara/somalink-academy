# SomaLink Academy

A responsive educational web application built using HTML5, CSS3, JavaScript, and Bootstrap.

## Project Overview

SomaLink Academy is a fictional online learning platform designed to help students gain practical skills in:

* Artificial Intelligence
* Data Science
* Web Development
* Financial Analysis

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
