# SomaLink Futures

## Empowering Africa's Next Generation of Innovators

SomaLink Futures is a responsive web platform designed to help students and young professionals discover career opportunities, access mentorship, explore scholarships, and develop future-ready skills.

## Project Objectives

The platform aims to:

* Connect students with industry mentors
* Provide curated learning pathways
* Share scholarship opportunities
* Promote professional certifications
* Showcase student success stories
* Support career growth through digital resources

## Features

* Responsive Bootstrap 5 design
* Multi-page website architecture
* Interactive navigation system
* Scholarship and career information hub
* Mentor directory
* Contact and support section
* FAQ section
* Dark mode support (planned)
* Search and filtering tools (planned)

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Git & GitHub
